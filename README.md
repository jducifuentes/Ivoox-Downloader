# Ivoox-Downloader
Download a list or audiobook from www.ivoox.com easily

## Requirements:
1. Python 3.6.4+
2. Beautiful Soup 4
3. unicodedata

## Instructions

`python ivoox.py url  path_to_save`

- Files will be save in path_to_save/name_of_program.  
- The script supports url with multiple pages.
